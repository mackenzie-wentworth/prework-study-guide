HTML

  • The head element contains information about the webpage
  • The information inside the head element is called metadata, which helps the browser to render the page correctly
  • Metadata inside the head element is typically not seen by users
  • The body element reperesents the visible content shown to the user
  • Elements that are nested inside other elements are called child elements
  • The element that holds all the children is known as the parent element

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicate how much space we want around the content inside an element
  • CSS rules are comprised of selectors and declarations
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain two important components: (1) the CSS property we want to apply and (2) the value of the propery

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: swtiches to a branch we want to move into next
  • git add -A: sets the stage for the changes you want to make

JavaScript

  • JavaScript allows you to power your websites with effects, interactivity, data communication, and much more.
  • JavaScript is case sensitve.
  • Variables, strings, and data types are the basic features of JavaScript.
  • A variable is a named container that allows us to store data in our code
  • Variables may hold values that have different data types.
  • Arrays are structures that allow you to store multiple values in a single reference.
  • Control flow is the order in which a computer executes code in a script
